Updates and plans

Posted on: March 19, 2026

I’ve been slowly working on the Discord bot updates. It’s taking some time as I’ve been navigating a bit of a creative block with it lately, but I'm still pushing forward.

In better news, the website finally has its modern redesign! It took 7 months, but every page has been officially redone.

I’ve uploaded my CC:Tweaked system code to GitHub. If you use that mod, you can just copy-paste the scripts and you’re good to go.

The server has been running for 1 year, 8 months, and 29 days (over 45k in-game days!). Just a heads-up: this site is for lore only; the server remains private for me and my friends.

Beyond the bot updates, I’m starting some new coding projects—including a Minecraft mod. I’ve also started writing a book; I’ve had some great ideas lately and want to see where they go.
